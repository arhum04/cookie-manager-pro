# Security Policy

## Supported Versions

The project is currently in active development. Only the latest version receives security updates.

## Reporting a Vulnerability

Security is a top priority for this extension as it handles sensitive cookie data.

If you discover a security vulnerability, please do not open a public issue. Instead, report it by opening a private security advisory on GitHub or by contacting the maintainer directly through their GitHub profile.

Reports will be acknowledged within 48 hours, and a timeline for resolution will be provided.
